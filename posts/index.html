Vulkan Compute Example

 2021-07-07 2471 words 12 minutes vulkan vulkan-compute GPGPU 
A Simple Vulkan Compute Example in C++ Vulkan is great. It provides a cross-platform API to write applications that use the GPU to do graphics and general purpose compute. Designed from the ground-up to be a modern API, using Vulkan can be quite difficult so you better know what you’re doing if you plan to use Vulkan for your application. Vulkan provides both a graphics and compute APIs and in this post I will focus on the compute part of it as I’m still not very familiar with the graphics side of it.
